# 制造业细分案例包：电子装配 - ECO频繁 + BOM版本错用 + 追溯困难

## 结论
电子装配企业这类问题，核心通常不是“现场不认真”，而是 **ECO/ECR 变更治理、BOM版本生效控制、料站防错、序列号追溯** 没有真正闭环。

## 典型现状
- 设计频繁变更，工厂经常追不上
- 同一产品不同版本混线风险高
- 客诉后很难快速追到批次、工单、料站、测试记录
- MES 有数据，但跨系统追溯仍慢

## 关键问题
1. ECO 生效规则不清
2. 生产版本切换控制弱
3. 序列号、批次、测试、返修数据链不完整
4. 客诉和工厂追溯脱节

## 一期建议
1. 建 ECO/BOM/工艺版本生效规则
2. 建关键工位防错与料站校验
3. 建序列号+批次+测试记录追溯链
4. 抓高频客诉专题复盘
