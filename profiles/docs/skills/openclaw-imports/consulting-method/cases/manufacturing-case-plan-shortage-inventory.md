# 制造业组合案例包：计划失真 + 缺料 + 高库存并存

## 结论
这类问题表面是计划不准，实质通常是 **预测机制失效、计划分层缺失、主数据不稳、变更纪律失控、供应协同弱** 共同导致。

## 典型现状
- 销售说客户催货，计划说插单太多，采购说供应商来不及，仓库说库存很多但关键料总缺
- 月度库存金额高，但产线仍频繁待料
- MRP 经常重跑，结果仍然不可信

## 问题拆解
### 业务层
- 需求承诺与产能现实脱节

### 流程层
- S&OP、MPS、MRP、排产未分层
- 插单与变更缺少冻结区

### 系统/数据层
- BOM、提前期、安全库存、替代料规则不准
- 在途、在制、齐套状态不能可信回写

### 供应协同层
- 关键供应商未共享真实节奏
- 关键物料缺乏风险预警

## 建议方案
### 一期
1. 建销售-计划-采购周例会机制
2. 建冻结区和插单升级规则
3. 治理关键物料计划参数
4. 抓关键供应商协同
5. 建缺料与高库存双清单复盘

### 二期
1. 推进 S&OP 机制
2. 推进关键供应商排产协同
3. 推进库存结构优化与替代料策略

## 风险提示
- 只压库存，不治结构
- 只重跑 MRP，不治参数
- 不管插单纪律，计划永远失真
