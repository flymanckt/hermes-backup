# 制造业专题包：电子装配 / SMT / PCBA / 返修追溯

## 适用场景
适用于电子装配、PCBA、整机装配企业，关注：
- SMT / DIP / 组装 / 测试 / 包装全流程
- 物料齐套与替代料控制
- 批次/序列号追溯
- 良率、直通率、返修率
- ECR/ECO 与版本切换

## 常见痛点
1. 小料多、替代料多，齐套难
2. 工单频繁切换，现场混料风险高
3. 不良定位慢，返修闭环弱
4. 追溯链路不完整，客诉追批次困难
5. BOM/ECO 变更多，现场版本错用风险高

## 方案抓手
- 强化料站/批次/序列号追溯
- 强化替代料审批与版本控制
- 强化首件、AOI、测试、返修闭环
- 强化工单切换与混料防错
- 强化良率/直通率/返修率联动分析

## 系统边界建议
- ERP：BOM、工单、库存、成本主线
- MES：过站、采集、序列号、测试、返修追溯
- WMS：批次、库位、发料、退料
- QMS：不良、CAPA、客诉闭环
