# 制造业专题包：主数据治理 / 物料 / BOM / 工艺 / 工作中心

## 适用场景
适用于以下问题：
- MRP 老跑偏
- BOM 经常错
- 工艺路线版本混乱
- 物料编码泛滥
- 计划、生产、成本、质量长期口径不一致

## 核心判断
制造业很多“系统问题”，根上其实是 **主数据治理失控**。没有稳定主数据，计划、执行、成本、质量都会一起失真。

## 核心对象
- 物料主数据
- BOM
- 工艺路线
- 工作中心/产线/设备
- 供应商
- 客户
- 仓库/库存地点
- 批次/序列号规则

## 常见痛点
1. 同物多码、一码多物
2. BOM 和现场实际不一致
3. 工艺路线维护随意
4. 工作中心能力参数失真
5. 主数据变更没人负责

## 方案框架
### 1. 明确主数据归口责任
### 2. 建主数据申请/审批/变更机制
### 3. 建版本与生效日期管理
### 4. 建主数据质量指标
### 5. 建主数据与计划/制造/成本/质量联动规则

## 一期建议
1. 先抓物料/BOM/工艺三大核心对象
2. 先统一编码规则和归口责任
3. 先治理关键产品线主数据
4. 先建立版本和生效控制

## 高风险点
- 只做台账整理，不做机制
- 变更无审批无追溯
- 主数据治理不和业务动作联动
