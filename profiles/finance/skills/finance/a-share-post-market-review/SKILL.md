# A股盘后复盘报告生成

## 触发条件
- 每日 15:05 执行（A股收盘后5分钟）
- 手动触发：`盘后复盘` / `daily review`

## 数据获取

### 指数数据（稳定）
使用 EastMoney 指数 API：
```
https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&invt=2&fields=f1,f2,f3,f4,f12,f14&secids=1.000001,0.399001,0.399006
```
返回字段：f2=最新价, f3=涨跌幅, f14=指数名称

### 个股数据（稳定方案）
**⚠️ EastMoney 个股 API (`push2.eastmoney.com/api/qt/stock/get`) 常断连，不要使用！**

改用 **Sina Finance**：
```
https://hq.sinajs.cn/list=sh603341,sz002969,sh603316,sh560390
```
- URL 参数格式：`sh` + 沪市代码，或 `sz` + 深市代码
- 返回 GBK 编码，需 `decode('gbk')`
- 格式：`var hq_str_sh603341="龙旗科技,开盘价,昨日收盘,当前价,最高,最低,成交量(股),..."`

字段顺序（以0起始）：0=名称, 1=开盘, 2=昨日收盘, 3=当前, 4=最高, 5=最低, 6=成交量

### 涨停价计算
```
涨停价 = 前收盘 × 1.10  （主板10%）
涨停价 = 前收盘 × 1.20  （创业板/科创板）
```

## 持仓数据结构
```python
holdings = {
    "603341": {"name": "龙旗科技", "cost": 45.551, "qty": 100},
    "002969": {"name": "嘉美包装", "cost": 24.840, "qty": 300, "stop": 23.50},
    ...
}
```

## 报告结构

1. **大盘收盘情况**：上证/深证/创业板 指收盘点位和涨跌幅
2. **持仓逐个复盘**：
   - 收盘价 vs 成本，涨跌幅，浮亏浮盈
   - 开盘/最高/最低/收盘 + 成交量
   - 走势回顾（具体描述）
   - 止损位距离（⚠️ 标注已破止损的标的）
3. **持仓表现归因**：跑赢/跑输大盘多少 pp
4. **问题诊断**：哪些判断正确，哪些有问题
5. **明日操作预案**：重点标的 + 具体操作清单

## 关键警示逻辑

```python
# 止损判断
if current_price <= stop_loss_price:
    print("⚠️ 止损已被击穿！止损机制失效，需手动决策")
elif current_price <= stop_loss_price * 1.05:
    print("🔴 逼近止损位，密切关注")

# 止损已破 → 不再作为持股依据，以明日开盘走势决策
```

## 常见错误

- **把"冲高回落"误判为"涨停"**：需对照昨日收盘计算10%/20%涨停价，不要凭感觉
- **Sina API 用UTF-8解码**：返回的是 GBK 编码，必须 `decode('gbk')`
- **ETF 代码拼接**：ETF 沪市用 `sh` 前缀，如 `sh560390`

## 输出
- 直接打印报告文本，全程中文
- 止损风险用**加粗**
- 发送到当前会话（cronjob 自动投递）
