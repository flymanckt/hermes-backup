# A股行情获取 — 腾讯gtimg备用方案

## 触发条件
当新浪hq.sinajs.cn返回数据不完整（昨收为0）、EastMoney push2 API无响应、Xueqiu被拦截时，使用此方案。

## 数据源
**腾讯行情API（无需注册，直接调用）**
```
https://qt.gtimg.cn/q=<市场前缀><股票代码>
```

- 市场前缀：`sh`=上交所 `sz`=深交所
- 示例：`https://qt.gtimg.cn/q=sh603341,sz002969,sh603316,sh560390`

## 字段解析（v_shXXXXXX格式）

字段位置从0开始，关键字段：
| 位置 | 字段名 | 示例(龙旗科技) |
|------|--------|---------------|
| 1 | 名称 | 龙旗科技 |
| 3 | 当前价格 | 38.29 |
| 4 | 昨收价格 | 38.76 |
| 5 | 今开 | 38.76 |
| 6 | 成交量(手) | 92751 |
| 9-14 | 卖1-5档 | 价格 |
| 19-24 | 买1-5档 | 价格 |
| 32 | 最高 | 38.30 |
| 33 | 最低 | 37.13 |
| 34 | 涨跌额 | -0.47 |
| 35 | 涨跌幅% | -1.21 |
| 36 | 总市值(万) | 35065 |

## 指数获取（同步）
```
https://qt.gtimg.cn/q=s_sh000001,s_sz399001,s_sz399006
```
返回格式：`上证指数,当前点位,涨跌,涨跌幅%,成交量...`

## 快速验证命令
```bash
curl -s "https://qt.gtimg.cn/q=sh603341,sz002969,sh603316,sh560390" | head -c 3000
```

## 优缺点
- ✅ 无需登录/注册/UA伪装，直接可用
- ✅ 数据完整（含昨收、可计算涨跌幅）
- ✅ 速度快
- ❌ 字段顺序固定，需对照文档解析
- ❌ 无基本面数据（PE/换手率等需另取）

## 已知失效情况
- 新浪hq.sinajs.cn的昨收字段在集合竞价时返回0，但其他数据正常
- EastMoney push2有时超时，换用此源即可
