# 制造业专题包：仓储 / 物流 / 账实一致 / 批次追溯

## 适用场景
适用于以下问题：
- 库存账实不一致
- 线边库混乱
- 盘点耗时长、差异大
- 批次追溯慢
- 仓储作业拖慢生产节奏

## 核心判断
仓储问题不是单纯仓库效率问题，而是 **库存状态、库位纪律、发料补料机制、批次追溯、系统执行一致性** 的综合问题。

## 方案框架
### 1. 建库存状态模型
### 2. 建库位与线边库规则
### 3. 建收货/上架/拣料/发料/退料/盘点闭环
### 4. 建批次/序列号追溯机制
### 5. 建账实一致率指标和差异治理机制

## 系统边界建议
- ERP：库存主账、单据主线
- WMS：仓储执行、库位、批次、盘点
- MES：线边领退补、生产领料协同

## 一期建议
1. 先梳理库存状态和库位规则
2. 先抓关键仓和线边库试点
3. 先统一批次追溯规则
4. 先建立账实差异复盘机制

## 高风险点
- 只做系统上线，不改仓储纪律
- 批次规则不清却要求快速追溯
- 线边库长期脱离系统管理
